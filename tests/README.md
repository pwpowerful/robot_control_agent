# Tests Directory

This directory contains all test-related code for the robotic arm control agent system.

## Responsibilities
- Unit Tests: Test individual modules, functions, and classes.
- Integration Tests: Test interactions between modules, such as API calls and database operations.
- End-to-End Tests: Test full workflows via Playwright for the web console.
- Test Fixtures: Setup data and mocks for tests.
- Test Configuration: Environment and settings for running tests.

## Structure
- `unit/`: Unit test files.
- `integration/`: Integration test files.
- `e2e/`: End-to-end test files (Playwright).
- `fixtures/`: Test data and mocks.
- `conftest.py`: Pytest configuration.

No business implementation in this step; only structure setup.
